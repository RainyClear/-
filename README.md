# 语法
## 1.输入/输出函数
### 写输入/输出 =>输入输出的内容即可输入或者输出
### tips =>左边可以随意空格，右边不可以，右边的内容不用管数据类型，模块会自动判断

## 2.定义变量
### 语法: 变量:变量名:内容
### tips :的左边可以随意空格，右边不可以，第一个冒号左右两边都可以

## 3.判断
### 语法: 
```python
如果:条件
    成立代码
否则
    不成立代码
```
### tips :左边可以空格，右边不可以
